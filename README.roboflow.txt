
rust-ai-bot - v1 2020-06-12 4:41pm
==============================

This dataset was exported via roboflow.ai on June 12, 2020 at 11:43 PM GMT

It includes 775 images.
Rust-game-objects-bot are annotated in YOLO v5 PyTorch format.

The following pre-processing was applied to each image:
* Auto-orientation of pixel data (with EXIF-orientation stripping)
* Resize to 608x608 (Fit within)

No image augmentation techniques were applied.
